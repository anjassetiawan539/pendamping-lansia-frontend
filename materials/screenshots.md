## üë®‚Äçüíº Admin Features

### 1. Dashboard Admin

![Dashboard Admin](ss-admin/Dashboard%20Admin.png)

**Deskripsi:**
Dashboard utama untuk administrator yang menampilkan overview sistem Teman Lansia. Fitur-fitur yang ditampilkan meliputi:

- **Statistik keseluruhan**: Total users (lansia & relawan), total requests, dan completed services
- **Grafik aktivitas**: Visualisasi tren penggunaan platform
- **Recent activities**: Daftar aktivitas terbaru di sistem
- **Quick actions**: Akses cepat ke fitur-fitur utama admin

Dashboard ini memberikan gambaran menyeluruh tentang performa dan aktivitas platform.

---

### 2. Tambah User lewat Admin

![Tambah User lewat Admin](ss-admin/Tambah%20User%20lewat%20Admin.png)

**Deskripsi:**
Halaman untuk menambahkan user baru ke dalam sistem oleh administrator. Fitur yang tersedia:

- **Form input lengkap**: Username, email, password, nama lengkap, nomor telepon
- **Role selection**: Pilihan role (ADMIN, LANSIA, atau RELAWAN)
- **Profile details**: Input kota, provinsi, alamat detail, dan bio
- **Validasi form**: Validasi real-time untuk memastikan data valid

Administrator dapat dengan mudah mendaftarkan user baru tanpa melalui proses registrasi publik.

---

### 3. Edit User oleh Admin

![Edit User oleh Admin](ss-admin/Edit%20User%20oleh%20Admin.png)

**Deskripsi:**
Interface untuk mengedit informasi user yang sudah terdaftar. Kemampuan admin meliputi:

- **Update profil lengkap**: Mengubah semua field informasi user
- **Change role**: Mengubah role user jika diperlukan
- **Status management**: Aktivasi atau deaktivasi akun user
- **History tracking**: Melihat riwayat perubahan data user

Fitur ini memungkinkan admin untuk memelihara data user tetap akurat dan up-to-date.

---

### 4. Edit Profil Admin

![Edit Profil Admin](ss-admin/Edit%20Profil%20Admin.png)

**Deskripsi:**
Halaman khusus untuk administrator mengedit profil mereka sendiri. Fitur yang tersedia:

- **Personal information**: Update nama, email, nomor telepon
- **Profile picture**: Upload atau ubah foto profil
- **Location details**: Update kota, provinsi, dan alamat
- **Bio & preferences**: Mengatur bio dan preferensi akun
- **Change password**: Opsi untuk mengubah password

Admin dapat mengelola profil mereka sendiri secara terpisah dari user lain.

---

### 5. Hapus User oleh Admin

![Hapus User oleh Admin](ss-admin/Hapus%20User%20oleh%20Admin.png)

**Deskripsi:**
Modal konfirmasi untuk menghapus user dari sistem. Fitur keamanan meliputi:

- **Confirmation dialog**: Dialog konfirmasi untuk mencegah penghapusan tidak sengaja
- **User information display**: Menampilkan detail user yang akan dihapus
- **Warning message**: Peringatan bahwa aksi ini permanen
- **Cancel option**: Opsi untuk membatalkan aksi penghapusan

Fitur ini dilengkapi dengan safeguard untuk melindungi data user dari penghapusan yang tidak disengaja.

## üë¥ Lansia (Elderly) Features

### 1. Dashboard Lansia

![Dashboard Lansia](ss-lansia/Dashboard%20Lansia.png)

**Deskripsi:**
Dashboard khusus untuk pengguna lansia yang menampilkan informasi personal dan layanan. Fitur utama:

- **Profil overview**: Informasi pribadi lansia dan status akun
- **Active requests**: Daftar permintaan layanan yang sedang aktif
- **Request history**: Riwayat layanan yang pernah diminta
- **Available services**: Layanan-layanan yang tersedia
- **Quick request**: Tombol cepat untuk membuat permintaan baru
- **Statistics**: Jumlah layanan yang sudah digunakan dan rating rata-rata

Dashboard dirancang user-friendly untuk kemudahan akses lansia.

---

### 2. Menu Permintaan Layanan oleh Lansia

![Menu Permintaan Layanan oleh Lansia](ss-lansia/Menu%20Permintaan%20Layanan%20oleh%20Lansia.png)

**Deskripsi:**
Halaman untuk membuat dan mengelola permintaan layanan. Fitur yang disediakan:

- **Request form**: Form untuk membuat permintaan baru dengan field:
  - Jenis layanan (Belanja, Bantuan Medis, Teman Ngobrol, Antar Jemput)
  - Deskripsi detail kebutuhan
  - Tanggal dan waktu yang diinginkan
  - Estimasi durasi layanan
- **Request list**: Daftar semua permintaan dengan status masing-masing
- **Status indicators**: Visual indicator untuk status (OFFERED, ASSIGNED, ON_GOING, DONE, CANCELLED)
- **Action buttons**: Edit, cancel, atau delete request

Lansia dapat dengan mudah membuat dan mengelola permintaan layanan sesuai kebutuhan mereka.

---

### 3. Modal Form Edit Profil Lansia

![Modal Form Edit Profil Lansia](ss-lansia/Modal%20Form%20Edit%20Profil%20Lansia.png)

**Deskripsi:**
Modal popup untuk mengedit profil lansia. Fitur yang tersedia:

- **Editable fields**: Nama lengkap, nomor telepon, email
- **Location update**: Kota, provinsi, dan alamat lengkap
- **Bio section**: Area untuk menuliskan informasi tambahan
- **Profile picture**: Upload atau ganti foto profil
- **Validation**: Validasi form sebelum submit
- **Cancel & Save**: Tombol untuk membatalkan atau menyimpan perubahan

Modal dirancang simpel dan mudah digunakan untuk lansia yang mungkin kurang familiar dengan teknologi.

### 4. Halaman Rating dan Ulasan oleh Lansia ke Relawan

![Halaman Rating dan Ulasan oleh Lansia ke Relawan](ss-lansia/Halaman%20Rating%20dan%20Ulasan%20oleh%20Lansia%20ke%20Relawan.png)

**Deskripsi:**
Interface untuk memberikan rating dan review kepada relawan setelah layanan selesai. Fitur meliputi:

- **Star rating**: Rating 1-5 bintang untuk kualitas layanan
- **Comment box**: Area untuk menulis review detail
- **Service details**: Informasi layanan yang sudah diselesaikan
- **Volunteer info**: Nama dan foto relawan yang memberikan layanan
- **Tags selection**: Pilihan tag seperti "HELPFUL", "PUNCTUAL", "FRIENDLY", "PROFESSIONAL"
- **Submit button**: Kirim review dan rating

Fitur ini membantu membangun sistem reputasi dan feedback untuk relawan.

## üë®‚Äç‚öïÔ∏è Relawan (Volunteer) Features

### 1. Dashboard Relawan

![Dashboard Relawan](ss-relawan/Dashboard%20Relawan.png)

**Deskripsi:**
Dashboard untuk relawan yang menampilkan aktivitas dan statistik mereka. Fitur utama:

- **Profile summary**: Informasi profil relawan dan rating
- **Available requests**: Daftar permintaan layanan yang tersedia untuk diambil
- **My tasks**: Tugas layanan yang sedang aktif atau telah di-assign
- **Statistics**: Total layanan yang diselesaikan, rating rata-rata, total jam bantuan
- **Earnings/Points**: Sistem poin atau reward (jika ada)
- **Quick actions**: Akses cepat untuk melihat tugas baru

Dashboard membantu relawan mengelola waktu dan komitmen mereka dengan efektif.

---

### 2. Halaman Tugas Layanan untuk Relawan

![Halaman Tugas Layanan untuk Relawan](ss-relawan/Halaman%20Tugas%20Layanan%20untuk%20Relawan.png)

**Deskripsi:**
Halaman yang menampilkan daftar tugas layanan untuk relawan. Fitur meliputi:

- **Task list**: Daftar semua tugas dengan status berbeda:
  - **Available**: Request yang bisa di-accept
  - **Assigned**: Request yang sudah di-accept, belum dimulai
  - **On Going**: Request yang sedang berlangsung
  - **Completed**: Request yang sudah selesai
- **Task details**: Informasi detail setiap request (nama lansia, lokasi, waktu, durasi)
- **Action buttons**:
  - Accept (untuk status OFFERED)
  - Start (untuk status ASSIGNED)
  - Complete (untuk status ON_GOING)
- **Filter & search**: Memfilter tugas berdasarkan status atau lokasi

Relawan dapat dengan mudah melihat dan mengelola semua tugas mereka dari satu halaman.

---

### 3. Halaman Lihat Rating dan Ulasan dari Lansia

![Halaman Lihat Rating dan Ulasan dari Lansia](ss-relawan/Halaman%20Lihat%20Rating%20dan%20Ulasan%20dari%20Lansia.png)

**Deskripsi:**
Halaman untuk relawan melihat rating dan review yang mereka terima dari lansia. Fitur meliputi:

- **Overall rating**: Rating rata-rata dari semua review
- **Total reviews**: Jumlah total review yang diterima
- **Rating breakdown**: Distribusi rating (berapa banyak 5 bintang, 4 bintang, dst.)
- **Review list**: Daftar semua review dengan:
  - Nama reviewer (lansia)
  - Rating bintang
  - Komentar/feedback
  - Tanggal review
  - Jenis layanan yang direview
- **Tags summary**: Ringkasan tag yang paling sering diberikan
- **Filter options**: Filter berdasarkan rating atau tanggal

Halaman ini membantu relawan memahami feedback mereka dan meningkatkan kualitas layanan.
