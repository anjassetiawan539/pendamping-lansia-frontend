# Pendamping Lansia - Web Frontend

Ini adalah **web frontend** untuk proyek **Pendamping Lansia**, bagian dari proyek kelompok.
Frontend dibangun menggunakan **HTML, CSS, dan JavaScript murni**.

## Kontribusi Saya
- Membuat **halaman dashboard dan UI interaktif**
- Implementasi **CRUD data lansia**
- **Session & Authentication Management** menggunakan JavaScript
  - Menyimpan dan mengambil data user dari `sessionStorage`
  - Logout otomatis dan pembersihan token dari `localStorage`
  - Proteksi halaman dengan **Auth Guard** dan **Role Guard**
- Integrasi dengan backend API
- Membuat tampilan website responsif dan user-friendly
